# 🌌 2D N-Body Simulator (Python)

This project implements a simple 2D N-body simulation for interacting particles. It uses DIP to manage parameters, allowing default values, user overrides, and automatic computation of derived quantities.

## 🚀 Usage

Run the simulation:

```bash
python3 main.py
```

## 🧩 Overview

* Python: Numerical N-body solver
* DIP: Parameter definition, configuration, and derived values
* 2D Dynamics: Particle interactions in a plane

## ⚙️ Features

* Configurable simulation parameters via DIP
* Automatic handling of dependent/derived settings
* Lightweight and easy to extend for more advanced physics

## 📁 Output

Depending on configuration, the simulation can produce:
* particle trajectories
* diagnostic data for analysis or visualization
